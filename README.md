# Monaba

本アプリはmanabaのUI調整を行ったり、自動ログインを実装することでスマートフォンでも快適に使えるようにすることを目標に作られました  
現在、androidのみ対応してます ios/pcの方はmonaba.netへ  
また、現時点で対応している大学は千葉工業大学のみです
